## Descripción ✏️
Este es el proyecto final de la materia Programación Web (INF-224), se trata de una página web de un concecionario ficticio llamado _**TurboZone**_, en el cual se pide integrar una base de datos a una aplicación web. La base de datos está hecha en SQL Server Management Studio 20, y la aplicación web con **Blazor Web Assembly** utilizando el framework .NET 8.0 y el principio de **arquitectura limpia** en Visual Studio Community 2022.
